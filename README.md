# Monitor del Sistema Linux

## Trabajo Integrador
Materia: Arquitectura y Sistemas Operativos

## Descripción

Este proyecto consiste en el desarrollo de una aplicación en Python ejecutada dentro de una máquina virtual con Ubuntu Linux creada mediante Oracle VirtualBox.

El objetivo es demostrar el uso de la virtualización como entorno de desarrollo, implementando una aplicación capaz de consultar información del sistema operativo, como memoria RAM, espacio en disco, procesos activos, información del sistema y otros recursos.

## Objetivos

- Comprender el funcionamiento de la virtualización.
- Utilizar Oracle VirtualBox para crear una máquina virtual.
- Configurar Ubuntu como sistema operativo invitado.
- Desarrollar una aplicación en Python.
- Obtener información del sistema operativo mediante comandos de Linux.
- Demostrar un caso práctico de virtualización aplicada al desarrollo de software.

## Tecnologías utilizadas

- Oracle VirtualBox
- Ubuntu Linux
- Python 3
- Visual Studio Code
